# 🎮 Танки на минималках 2.0

Классическая аркадная игра «Танчики» на Python с использованием Tkinter. Управляйте танком, уничтожайте врагов и собирайте бонусы на процедурно-генерируемой карте.

## 📋 Содержание

- [Особенности](#-особенности)
- [Управление](#-управление)
- [Игровая механика](#-игровая-механика)


## ✨ Особенности

- **Управление игроком** — движение W/A/S/D, стрельба пробелом
- **ИИ противники** — вражеские танки с логикой преследования и случайным движением
- **Разрушаемое окружение** — кирпичные стены можно пробивать снарядами
- **Бонусы** — подбирайте ящики с боеприпасами (+10 к боезапасу)
- **Динамическая камера** — слежение за игроком с возможностью ручного управления
- **Физика столкновений** — танки не проходят сквозь стены и друг друга
- **Различные типы поверхностей** — вода замедляет движение танков
- **Система боезапаса** — ограниченное количество снарядов
- **Топливная система** — ограниченный запас хода

## 🕹️ Управление

| Действие | Клавиша |
|----------|---------|
| Вперёд | `W` |
| Назад | `S` |
| Влево | `A` |
| Вправо | `D` |
| Стрельба | `Пробел` |

## 🎯 Игровая механика

### Танки
- **Игрок** — управляемый танк с красными текстурами
- **Враги** — танки с ИИ (голубые текстуры)
- **Характеристики:**
  - HP: 100 единиц
  - Боезапас: 80-100 снарядов
  - Скорость: зависит от типа поверхности
  - Топливо: 10000 единиц

### Снаряды
- Летят в направлении движения танка-владельца
- Разрушают кирпичные стены (один выстрел - один блок)
- Останавливаются о бетонные стены
- Игнорируют воду и землю
- Уничтожают танк при попадании

### Карта
**Размеры:**
- Ширина: 4800 пикселей (6 экранов)
- Высота: 3200 пикселей (4 экрана)
- Размер блока: 64×64 пикселя

**Типы блоков:**
| Блок | Обозначение | Эффект |
|------|-------------|--------|
| 🌿 Земля | GROUND | Обычное движение |
| 💧 Вода | WATER | Замедление движения (50% скорости) |
| 🧱 Кирпич | BRICK | Разрушается от выстрелов |
| 🏢 Бетон | CONCRETE | Неразрушаемый |
| 📦 Бонус | MISSLE | +10 к боезапасу |

### Искусственный интеллект
- 1 раз в 30 кадров бот принимает решение
- С вероятностью 90% движется к игроку
- С вероятностью 10% меняет направление случайно
- При столкновении автоматически меняет направление
